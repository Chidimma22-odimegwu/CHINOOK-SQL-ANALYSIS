# Chinook Music Store SQL Analysis

##  Project Overview
This project analyzes the Chinook digital music store database using SQL Server (SSMS).  
The goal is to extract business insights related to sales performance, customer behavior, product popularity, and executive KPIs.
The analysis progresses from basic SQL queries to advanced analytics and executive-level reporting.

##  Tools Used
- SQL Server Management Studio (SSMS)
- SQL (T-SQL)
- Chinook Sample Database

##  Dataset
The Chinook database simulates a digital music store containing:
- Artists
- Albums
- Tracks
- Customers
- Invoice
- Playlists
- Genres
- employee
- playlisttrack
- Invoiceline
- Mediatype

##  Analysis Structure
### Tier 1 — Basic Exploration
- Artist and album analysis
- Genre distribution
- Customer overview

### Tier 2 — Sales & Customer Behavior
- Unsold tracks
- Top-selling artists
- Customer purchase patterns
- Revenue by country

### Tier 3 — Advanced Analytics
- Running totals
- Popularity metrics
- Playlist analysis
- Window functions

### Tier 4 — Executive Reporting
- Annual revenue trends
- Customer lifetime value
- KPI dashboard queries
- CEO summary metrics

##  Key Business Questions Answered
- Which artists generate the most revenue?
- What genres drive sales?
- Who are the highest-value customers?
- Which tracks were never purchased?
- How does revenue trend over time?

## Skills Demonstrated
- SQL Joins
- Aggregations
- Window Functions
- CTEs
- Business KPI Analysis
- Data Modeling Understanding
- Query Optimization
